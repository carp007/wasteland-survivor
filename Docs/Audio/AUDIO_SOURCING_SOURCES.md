# Audio sourcing sources (free-first)

This is a short list of sources that are usually high-leverage for an indie project.

## Free / high value
- **Sonniss GDC GameAudio bundles** — huge, professional SFX drops (explosions, impacts, ambience). Verify the license for each yearly bundle and record the bundle name + year in the manifest.
- **Kenney audio packs** — great CC0 UI + general SFX; fast placeholders that often ship.
- **Freesound** — massive library; prefer CC0. CC-BY is OK if we track attribution. Avoid NC.
- **Pixabay SFX** — easy filler; still track source + license notes.
- **OpenGameArt** — quality varies; useful for placeholders.

## Low-cost paid (spend where it matters)
Engines are “hero sounds” in this game. It’s often worth spending a little money on:
- a **vehicle engine bundle** (multiple archetypes)
- a few **signature weapon sets**

If we buy a pack, record:
- store page link
- license terms / proof of purchase reference
- any redistribution restrictions

## Practical tips
- For **engine realism**, prefer multi-layer loop sets (idle/low/mid/high) over a single loop.
- Keep **raw downloads** in `_INBOX/` and never reference them from game scenes.
- Always create 2–5 **variants** for repetitive one-shots (clicks, impacts, small explosions) to avoid fatigue.
